# Анализ резюме HeadHunter

Учебный проект по исследовательскому анализу данных: обработка и визуализация резюме соискателей с платформы HeadHunter.

## Описание

В ноутбуке выполняется полный цикл работы с данными:

1. **Исследование структуры** — загрузка датасета, первичный обзор признаков.
2. **Преобразование данных** — извлечение и нормализация признаков:
   - образование, пол и возраст;
   - опыт работы (в месяцах);
   - город, готовность к переезду и командировкам;
   - тип занятости и график работы;
   - заработная плата в рублях с учётом курса валют.
3. **Анализ зависимостей** — распределения возраста, опыта и зарплаты; связь зарплаты с образованием, городом и готовностью к переезду.
4. **Очистка данных** — удаление дубликатов и аномальных значений зарплаты.

## Структура проекта

```
HeadHunterProject/
├── Project-1._Ноутбук-шаблон.ipynb   # основной ноутбук с анализом
├── dst-3.0_16_1_hh_database.csv      # резюме с HeadHunter (~44 700 записей)
├── ExchangeRates.csv                 # курсы валют для пересчёта зарплаты
├── graphics/                         # экспортированные графики (PNG)
└── .gitattributes                    # настройки Git для ноутбуков
```

## Запуск

**Требования:** Python 3, Jupyter Notebook или JupyterLab.

```bash
pip install pandas numpy matplotlib seaborn plotly
jupyter notebook
```

Откройте `Project-1._Ноутбук-шаблон.ipynb` и выполните ячейки последовательно сверху вниз.

> CSV-файлы с данными должны лежать в корне проекта рядом с ноутбуком. В `.gitignore` они не отслеживаются — скачайте их отдельно, если клонируете репозиторий.

## Используемые библиотеки

| Библиотека   | Назначение                          |
|--------------|-------------------------------------|
| pandas       | загрузка и обработка таблиц         |
| numpy        | численные операции                  |
| matplotlib   | базовые графики                     |
| seaborn      | статистическая визуализация         |
| plotly       | интерактивные диаграммы             |

## Данные

CSV-файлы не хранятся в репозитории (см. `.gitignore`). Скачайте их по ссылкам:

- [dst-3.0_16_1_hh_database.csv](https://drive.google.com/file/d/1_Dz8EPQ3t1Rruhtp-MWwSHkr_k9tksfP/view?usp=sharing) — анонимизированные резюме (~44 700 записей); разделитель полей — `;`.
- [ExchangeRates.csv](https://drive.google.com/file/d/1aWCzCF53qk4cO4wbZ5n8udprDrfyY_A2/view?usp=sharing) — исторические курсы валют для перевода желаемой зарплаты в рубли на дату обновления резюме.

## `.gitattributes`

В репозитории настроена нормализация переносов строк для ноутбуков:

```
*.ipynb text eol=lf
```

Это помогает избежать лишних diff-ов в `.ipynb` при работе на Windows и Linux/macOS: Git всегда сохраняет ноутбуки с LF-окончаниями строк.
